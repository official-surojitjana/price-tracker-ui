# 📊 Price Tracker Admin UI

A React-based admin dashboard for the **Price Tracker** platform.

This application provides an interface to monitor products, view price tracking data, and manage the automated product discovery system.

---

## ✨ Features

- 📦 View all tracked products
- 💰 Display latest product prices
- 🔍 Search products (Coming Soon)
- 📈 Product statistics (Coming Soon)
- ⚠️ Discovery failure monitoring (Coming Soon)
- 📊 Dashboard with platform metrics (Coming Soon)

---

## 🛠 Tech Stack

- React
- Vite
- Axios
- Tabler UI
- Bootstrap
- JavaScript (ES6)

---

## 🚀 Getting Started

### Install dependencies

```bash
npm install
```

### Start development server

```bash
npm run dev
```

The application will be available at:

```
http://localhost:5173
```

---

## 🔗 Backend

This UI communicates with the Spring Boot backend running on:

```
http://localhost:8080
```

Make sure the backend application is running before starting the UI.
